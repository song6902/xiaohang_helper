# 02 办事流程

> 维护人：小航项目组
> 更新日期：2026-07-15
> 数据来源：郑州航院学生手册、教务处官网、各部门办事指南

## 在校生常用办事流程

### 开具在读证明
- 办理地点：行政楼教务处窗口 ✏️ 待核实
- 所需材料：身份证、校园卡
- 办理时间：工作日 8:30-11:30，14:30-17:00 ⚠ 以官方为准
- 流程：
  1. 在自助打印机打印在读证明（图书馆、行政楼都有自助机）
  2. 如需盖章，到教务处窗口盖章
- 费用：免费
- 立等可取，不需要等

### 校园卡补办
- 办理地点：一卡通中心（后勤楼一楼）✏️ 待核实
- 所需材料：身份证
- 办理时间：工作日 9:00-16:00 ⚠ 以官方为准
- 费用：工本费20元 ⚠ 以官方为准
- 注意：
  1. 丢失后先在"郑航一卡通"公众号挂失，防止被人盗刷
  2. 补办后原卡余额自动转到新卡
  3. 补办后需要重新激活宿舍门禁、图书馆权限

### 转专业申请
- 申请时间：大一下学期开学后前两周 ⚠ 以官方为准
- 申请条件：
  1. 第一学期无挂科
  2. 成绩在专业前30% ✏️ 待核实
  3. 部分专业需要参加转专业考试
- 流程：
  1. 在教务系统提交转专业申请
  2. 转出学院签字同意
  3. 转入学院审核/考试
  4. 学校公示后正式生效
- 注意：转专业成功后需要补修新专业的课程

### 请假流程
- 请假1天以内：找辅导员签字批准
- 请假1-3天：辅导员签字 + 学院副书记批准
- 请假3天以上：学院批准 + 教务处备案
- 请假需要写请假条，注明请假原因、时间、去向
- 病假需要提供校医院或医院证明

### 奖学金申请
- 国家奖学金：每年9-10月申请，成绩专业前10%，8000元
- 国家励志奖学金：每年9-10月申请，家庭困难+成绩优秀，5000元
- 校级奖学金：每学期评，一等1000、二等600、三等300 ✏️ 待核实
- 流程：班级评议 -> 学院审核 -> 学校公示 -> 打款到银行卡

## 教师常用办事流程

### 调课申请
- 申请时间：至少提前3天在教务系统提交
- 所需材料：调课申请表，说明调课原因
- 流程：
  1. 在教务系统填写调课申请
  2. 系主任签字同意
  3. 教学副院长批准
  4. 教务处备案
- 注意：不能私自调课、私自找老师代课，必须走流程

### 多媒体设备报修
- 教室内多媒体坏了：直接打教室墙上的报修电话
- 或者联系：现代教育技术中心 0371-6191xxxx ⚠ 以官方为准
- 紧急情况可以直接找教学楼值班室，会有老师过去处理

### 差旅费报销
- 报销流程：
  1. 出差前在OA系统提交出差申请
  2. 出差回来后整理发票（车票、住宿票、会议通知）
  3. 在财务系统填写报销单
  4. 部门领导签字
  5. 财务处审核打款
- 标准：
  - 住宿：省内300元/天，省外350元/天 ⚠ 以官方为准
  - 交通：高铁二等座、火车硬座/硬卧
  - 伙食补助：100元/天 ⚠ 以官方为准

### 科研项目申报
- 国家级项目（国自然、国社科）：每年1-3月申报
- 省部级项目：每年3-5月申报
- 厅级项目：每年5-7月申报
- 申报入口：科研管理系统
- 咨询：科技处/社科处，各学院有科研秘书可以咨询

## 通用办事指南

### 教务处常用业务
- 成绩单打印：自助打印机免费打印，盖章在教务处窗口
- 学生证补办：每月15号统一办理，带身份证和一寸照片
- 缓考申请：考试前提交，需要医院证明或特殊情况说明
- 选课问题：先找教学秘书，解决不了再找教务处

### 后勤维修
- 宿舍水电、家具坏了：在"郑航后勤"公众号报修
- 或者打后勤维修电话：0371-6191xxxx ⚠ 以官方为准
- 紧急维修（水管爆了、电路坏了）直接打宿管阿姨电话

### 图书馆使用
- 开馆时间：8:00-22:00，考试周延长到23:00 ⚠ 以官方为准
- 借书：凭校园卡，本科生最多借10本，借期30天
- 续借：可以在图书馆公众号续借一次，延长30天
- 占座：图书馆有座位预约系统，不要用书本占座
