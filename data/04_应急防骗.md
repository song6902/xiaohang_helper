# 04 应急防骗

> 维护人：小航项目组
> 更新日期：2026-07-15
> 数据来源：郑州航院保卫处、学生处防骗手册、心理健康教育中心

## 紧急情况处理

### 遇到危险/盗窃/纠纷
1. 第一时间拨打校园110：0371-61916110（24小时有人接，比打110到校更快）
2. 保护好现场，不要和人正面冲突
3. 如果是宿舍盗窃，不要翻东西，等保卫处来查指纹
4. 人身安全第一，东西丢了可以再买，人没事最重要

### 火灾/消防
1. 小火：用灭火器、湿抹布扑灭，拔掉电源
2. 大火：立即跑出去，不要贪恋财物，走楼梯不要坐电梯
3. 打119 + 校园110：0371-61916110
4. 浓烟大的时候弯腰捂口鼻，贴墙走

### 突发疾病/受伤
1. 轻微受伤：去校医院（东苑食堂旁边）
2. 严重情况：打120 + 校园110，让保安来引路
3. 校医院急诊24小时有人：0371-6191xxxx ⚠ 以官方为准
4. 不要随便搬动重伤的人，等医生来

### 心理危机干预
⚠️ 如果你或者同学有以下想法，请立即求助：
- 觉得活着没意思、不想活了
- 情绪崩溃、控制不住自己
- 有伤害自己或别人的想法

求助渠道：
1. 全国心理援助热线：12320-5（24小时免费）
2. 学校心理健康教育中心：0371-6191xxxx ⚠ 以官方为准
3. 立刻告诉辅导员、班长、宿管阿姨
4. 陪着同学，不要让他一个人待着

## 新生防骗指南（重中之重！）

### 常见骗局1：冒充老师/辅导员收费
- 骗局："我是你辅导员，现在要交教材费/军训费/保险费，转我微信就行"
- 识别：
  ✅ 学校收费全部在官方系统交，不会让你转微信/支付宝给个人
  ✅ 辅导员不会私下加你要钱
  ✅ 任何要你现在立刻转钱的，都是骗子
- 做法：挂掉电话/不回消息，先在班级群@辅导员核实，或者打0371-61911000总机问

### 常见骗局2：上门推销
- 骗局："学长/学姐是英语协会的，订英语报纸四级必过，现在订送资料"
- 骗局："这是我们公司的护肤品/文具，便宜卖给你，支持一下学长创业"
- 识别：
  ✅ 真正的学长学姐不会上门推销东西
  ✅ 订报纸、买东西交了钱之后人就再也找不到了
  ✅ 四级资料网上几十块钱一大堆，不需要订报纸
- 做法：直接说"不需要"，关门就好，不要不好意思

### 常见骗局3：兼职刷单/打字员
- 骗局："足不出户日赚300，刷单/打字/点赞，一单一结"
- 套路：先给你点小甜头（前几单返5块10块），然后说要做联单，让你充钱，最后拉黑
- 识别：
  ✅ 刷单本身就是违法的
  ✅ 没有躺着赚钱的好事
  ✅ 凡是要你先充钱、垫钱的兼职，全是诈骗
- 做法：直接拉黑，不要信，不要贪小便宜

### 常见骗局4：助学金/奖学金退款
- 骗局："我是教育局的，你有一笔助学金/奖学金要发，给我银行卡号和验证码"
- 识别：
  ✅ 助学金、奖学金全部走学校官方渠道，不会有人给你打电话要验证码
  ✅ 任何要你验证码的，全是骗子
  ✅ 验证码就是钱，谁要都不给
- 做法：挂电话，找辅导员核实

### 常见骗局5：校园贷/培训贷
- 骗局："零利息借钱买手机/买电脑""培训包就业，先学后付"
- 套路：实际是高利贷，利滚利欠几万，还不上就威胁你
- 识别：
  ✅ 不要碰任何网络贷款
  ✅ 钱不够找爸妈、找辅导员，不要借网贷
  ✅ "先学后付""培训包就业"基本都是培训贷

## 老生防骗提醒

### 兼职防骗
- 找兼职走正规渠道（学校勤工助学中心、正规招聘APP）
- 凡是要交押金、服装费、培训费的，全是骗子
- 不要把银行卡、电话卡借给别人，别人可能用来洗钱
- 不要帮别人代收钱、转钱，可能是诈骗洗钱

### 网购/二手交易防骗
- 不要脱离平台交易，微信/支付宝直接转账的不要信
- 二手交易走闲鱼，不要点别人发的陌生链接
- "私下转你给你便宜点"基本都是骗子
- 快递丢了理赔，找官方客服，不要信主动打给你的电话

### 游戏/账号交易防骗
- 游戏账号、装备交易走正规平台
- "加QQ发你交易链接"的全是钓鱼网站
- 不要扫别人发的二维码付款

## 防骗黄金法则
1. **凡是让你转钱给个人的，一律是诈骗**
2. **凡是要你银行卡验证码的，一律是诈骗**
3. **凡是说"你涉嫌犯罪要你保密"的，一律是诈骗**
4. **凡是说"刷单赚钱、足不出户日赚几百"的，一律是诈骗**
5. **拿不准的，先问辅导员、问爸妈、打校园110，不要自己做决定**
6. **贪小便宜吃大亏，天上不会掉馅饼**

## 被骗了怎么办
1. 立刻打110或者校园110：0371-61916110报警
2. 保存好转账记录、聊天记录、对方账号，不要删
3. 立刻给银行打电话冻结银行卡
4. 告诉辅导员，不要觉得不好意思，很多人都被骗过
5. 第一时间报警还有可能把钱追回来
