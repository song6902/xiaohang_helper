# 小航 · 郑州航院校园信息查询AI助手

> 郑州航空工业管理学院 人工智能专业 大一认知实习项目

## 项目简介
"小航"是基于大语言模型的校园信息查询AI助手，专为郑州航院师生打造，支持新生、在校生、教师三类身份，提供入学指南、办事流程、电话黄页、应急防骗等校园信息查询服务。

## 技术栈
- **界面框架**：Streamlit
- **AI API**：硅基流动（SiliconFlow） - Qwen/Qwen2.5-7B-Instruct
- **核心技术**：Prompt工程 + API调用
- **数据格式**：Markdown

> 本项目理念：只用Prompt + API两样技术，不引入RAG、向量库、数据库、用户系统等复杂框架，符合大一认知实习定位。

## 快速开始
### 1. 安装依赖
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 2. 配置API Key
打开 `src/config.py`，把 `API_KEY` 替换成你自己的硅基流动API Key。

### 3. 运行项目
```bash
streamlit run src/app.py
```

## 核心功能
1. **身份分流**：新生/在校生/教师三套不同语气的Prompt
2. **智能问答**：基于学校资料回答校园相关问题
3. **推荐问题**：12个高频问题按钮，点击直接提问
4. **电话黄页**：静态兜底页，不依赖API，永远能查
5. **异常处理**：超时、断网、Key失效、数据缺失都有友好提示
6. **防幻觉机制**：6条硬规则，禁止编造信息，涉及金钱自动提示防骗

## 防幻觉硬规则
1. 只能根据学校资料回答，没有的明说没收录，建议打总值班室
2. 严禁编造电话/地址/时间/金额/人名
3. 涉及金钱/转账无条件提示防骗，先联系辅导员核实
4. 心理危机立即给援助热线+心理咨询中心+辅导员
5. 不接入学校系统，查个人信息礼貌拒绝
6. 回答末尾标注来源文件名
